TAREAS PENDIENTES - MINIMARKET CAMUCHA
========================================

1. FUNCIONALIDADES DEL ADMIN
   --------------------------
   - El admin va a enviar/aprobar pedidos
   - El admin va a enviar la confirmación del pago

2. AUTENTICACIÓN Y PERFIL DE USUARIO
   ----------------------------------
   - Implementar login (inicio de sesión)
   - Implementar perfil de usuario en el botón de perfil
   - Usar Better Auth como framework de autenticación
     * Repositorio: https://github.com/better-auth/better-auth
     * Framework-agnostic authentication para TypeScript
     * Incluye características avanzadas: 2FA, multi-tenant, OAuth, etc.
     * Documentación: https://better-auth.com

3. PANEL DE ADMINISTRACIÓN
   -----------------------
   - Revisar y desarrollar las funciones del panel admin
   - Implementar todas las funcionalidades necesarias del panel

NOTAS:
- Estas funcionalidades se implementarán en orden de prioridad
- Primero se trabajará en las funciones del admin (aprobación de pedidos y confirmación de pago)
- Luego se implementará el sistema de login y perfil de usuario usando Better Auth
- Finalmente se revisará y completará el panel de administración
- Better Auth es ideal para este proyecto ya que:
  * Está diseñado para TypeScript (compatible con nuestro stack)
  * Proporciona autenticación completa out-of-the-box
  * Tiene un ecosistema de plugins para funcionalidades avanzadas
  * Es framework-agnostic y funciona bien con Angular
