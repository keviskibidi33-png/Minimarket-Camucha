Configuración de Tarifas de Envío

Sistema simplificado: precio fijo para Lima con envío gratis a partir de un monto mínimo.

Precio fijo que se cobrará por envío a domicilio en Lima (sin importar distancia o peso)

Si el subtotal de la compra es igual o mayor a este monto, el envío será gratis

Cómo funciona:

  • Si el subtotal es menor a S/ {{ freeShippingThreshold().toFixed(2) }}, el envío cuesta S/ {{ fixedShippingPrice().toFixed(2) }}
  • Si el subtotal es igual o mayor a S/ {{ freeShippingThreshold().toFixed(2) }}, el envío es gratis
  • El retiro en tienda siempre es gratis

Ejemplos:

Compra de S/ 15.00: Envío: S/ {{ fixedShippingPrice().toFixed(2) }} | Total: S/ {{ (15 + fixedShippingPrice()).toFixed(2) }}
Compra de S/ {{ freeShippingThreshold().toFixed(2) }}: Envío: Gratis | Total: S/ {{ freeShippingThreshold().toFixed(2) }}
Retiro en tienda: Siempre gratis
