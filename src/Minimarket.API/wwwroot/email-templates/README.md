# Imágenes para Templates de Email

Esta carpeta contiene las imágenes utilizadas en los templates de email.

## Estructura

```
email-templates/
├── logo.png              # Logo de Minimarket Camucha (arriba del email)
├── promotion.png         # Imagen de promoción (abajo del email)
└── README.md            # Este archivo
```

## Cómo usar

Las imágenes se acceden mediante URLs:
- Logo: `https://tudominio.com/email-templates/logo.png`
- Promoción: `https://tudominio.com/email-templates/promotion.png`

## Nota

En desarrollo local, las imágenes estarán en:
- `http://localhost:5000/email-templates/logo.png`
- `http://localhost:5000/email-templates/promotion.png`
